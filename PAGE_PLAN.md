# 山禾个人网站 - 页面规划 v3

## 问题分析
- 当前所有内容都在一个页面，缺乏惊喜感
- 需要像那个樱花粉网站一样，分成多个独立页面
- 每个页面内容要更丰满

## 页面结构规划

### 1. 首页 (Home) - /index
- 简短的个人介绍
- 几个核心标签
- 导航到其他页面
- 动态粒子背景

### 2. 关于我 (About) - /about
- 详细个人简介
- 成长经历
- 兴趣爱好
- 照片展示区

### 3. 技能 (Skills) - /skills
- 技能分类展示
- 熟练度可视化
- 学习历程

### 4. 作品集 (Projects) - /projects
- 项目卡片展示
- 项目详情
- 技术栈说明

### 5. 日记/博客 (Blog) - /blog
- 文章列表
- 文章详情页

### 6. 联系 (Contact) - /contact
- 联系方式
- 社交链接
- 留言表单（可选）

---

## 设计风格
- 每个页面独立的背景和配色
- 保持MC像素风格
- 页面切换动画
- 丰富的装饰元素
- 响应式设计

## 技术实现
- 使用 Astro 多页面
- 共享 Layout 和 Components
- 每个页面独立样式
- 页面过渡动画
